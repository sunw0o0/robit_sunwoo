영화진흥위원회 역대 흥행 TOP 5 분석 프로그램

1. 사용 라이브러리
urllib3            2.7.0
requests           2.34.2

2. 구동 원리
   
(1) 이 프로그램은 인터넷망을 통해 원격에 있는 컴퓨터와 대화하는 클라이언트-서버 구조로 작동합니다.

(2)요청 (Request): 내 컴퓨터(Client)가 requests.get() 함수를 실행하는 순간, 전 세계 인터넷 통신 규약인 HTTP 프로토콜을 타고 영화진흥위원회 컴퓨터(Server)에 "역대 흥행 데이터 보내" 하고 신호를 보냅니다.

(3)응답 (Response): 요청을 받은 영진위 서버는 데이터베이스(DB)에서 영화 통계 자료를 꺼내와 가볍고 정돈된 텍스트 형식인 JSON 데이터로 내 컴퓨터에 돌려줍니다.

(4)역직렬화 (Deserialization): response.json() 코드가 텍스트로 된 JSON 데이터를 파이썬이 이해할 수 있는 딕셔너리({})와 리스트([]) 구조로 자동 변환해 메모리에 올립니다.

(5)예외 처리로 우회 (Try-Except): 에러가 감지되는 순간, except Exception 소포 상자가 실행되어 화면에 에러 대신 친절한 안내 문구를 띄우고, 미리 코드 내부에 심어둔 2026년 기준 역대 흥행 백업 데이터를 들고 복귀합니다.

(6)영구 백업 (File I/O): 실시간 데이터든 백업 데이터든 최종적으로 가공 완료된 결과물은 파이썬 파일 입출력(with open) 시스템을 통해 컴퓨터 하드디스크에 latest_query.json이라는 물리적인 파일로 안전하게 자동 저장(백업)됩니다.
